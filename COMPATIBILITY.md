# Running on old Android versions

If your device is running an older version of Android and cannot install the latest release, use the table below to download the newest compatible version of Persian Calendar for your device.

| Android version                                 | App version                                                                                                                                                                  |
| ----------------------------------------------- | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Android 6.0 (API 23) and up                     | [Persian Calendar (latest stable)](https://github.com/persian-calendar/persian-calendar/releases)                                                                            |
| Android 5.0 (API 21) to Android 5.1 (API 22)    | [Persian Calendar 10.0.0 (1000)](https://github.com/persian-calendar/persian-calendar/releases/download/v10.0.0/PersianCalendar-10.0.0-main-11493-4224a7e58-release.apk)     |
| Android 4.2 (API 17) to Android 4.4W (API 20)   | [Persian Calendar 8.4.0 (840)](https://github.com/persian-calendar/persian-calendar/releases/download/v8.4.0/PersianCalendar-8.4.0-main-7751-79d271919-minApi17-release.apk) |
| Android 4.0.3 (API 15) to Android 4.1 (API 16)  | [Persian Calendar 6.3.2 (632)](https://github.com/persian-calendar/persian-calendar/releases/download/v6.3.2/PersianCalendar-6.3.2-master-3208-65175c7c-release.apk)         |
| Android 2.3.0 (API 9) to Android 4.0.2 (API 14) | [Persian Calendar 5.4.0 (540)](https://github.com/persian-calendar/persian-calendar/releases/download/v5.4.0/PersianCalendar-release.apk)                                    |
| Android 2.1 (API 7) to Android 2.2 (API 8)      | [Persian Calendar 5.1.0 (510)](https://github.com/persian-calendar/persian-calendar/releases/download/v5.1.0/PersianCalendar-release.apk)                                    |
